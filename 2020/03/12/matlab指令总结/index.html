matlab指令总结

前言

  最初，我了解到matlab是由于参加数学建模竞赛的时候，在这学期的自动控制原理、信号与系统课程的时候会用到，所以原本我准备在上学的时候穿插着学一下matlab。但是在“传感器及检测技术”课程中，老师布置的作业里有一条需要使用matlab线性拟合，所以我就在寒假顺便把matlab看了一遍。
  我看了一本书–《matlab揭秘》郑碧波译，由于matlab本身是一个工具，所以指令有点多，不易记忆，所以，我准备写一个blog来记录一样，顺便总结一下，方便以后用的时候查询。
  前提说明一下，下面顺序和思路，都是仿照上面那本书所写，如果想学matlab可以自己看书学习，这里只是总结。

指令

MATLAB环境

命令窗口与算法基础

1 命令在”>>”提示符后面输入
2 如果未定义变量名，默认为ans

3 matlab输入基本算法：
两数相除，右除a/b，左除a\b=b/a
幂，a^b
相加，a+b
相减，a-b

赋值运算符

向量与矩阵

向量

向量(vector)一维数值数组。
1 列向量：
a=[2;1;4]
2 行向量：
v=[2 0 4] 或 v=[2,0,4]
3 转置 (‘)

从已存变量创建大向量

D=[A;B]

创建等差元素向量

x=[x:q:y] 或x=x:q:y
x为首元素，y为末元素，q为间隔

求矩阵的秩

rank(A) 表示矩阵A的秩

求逆矩阵与伪矩阵

inv(A) 可计算矩阵A的逆矩阵
pinv(A) 可计算矩阵A的伪逆矩阵

简化阶梯矩阵

rref(A) 使用高速-乔丹消元法产生矩阵A降阶后的阶梯形式
magic(n) 计算魔方矩阵是一个n*n矩阵，矩阵的元素从1到n平方之间，并且行元素的和等于列元素的和

矩阵分解

lu(A) 对矩阵A进行LU分解

绘图与图形

2D绘图基础

①在MATLAB中绘图包含下面三个步骤:
1.定义函数
2.指定要绘制的函数图形的值范围
3.调用 MATLAB的plot(x,y)函数
②[ start : interval : end ]表示stare<=x<=end,以interval递增
③通过xlabel,ylabel函数可以绘制你一个坐标轴的标签；通过title函数添加标题
④fplot 函数会绕过绘图的时间间隔，自动决定绘图的点数


MATLAB图像中可以使用四种基本线条风格：
实线 ‘ - ‘ 虚线’ – ‘ 虚点线 ‘ -. ‘ 点线 ‘ : ‘

添加图例

legend加在plot命令后面，用单引号把你要添加为图例的文本引起来

子图

subplot(m,n,p)表示产生的子图有m行和n列，p表示要贴上去的某个已经绘制的图形窗口

图像重叠和linspace命令

linspace命令用来产生x数集的新命令


loglog 产生整个数据范围内的函数
semilogx产生图像x轴使用对数值，y轴仍然用直接值
semilogy产生图像y轴使用对数值，x使用直接值

离散数据绘图

bar 能用二维条形图把数据显示出来
stem 绘制成针头图
可以通过传递fill参数选项让MATLAB填充标记，其中包括方块(s)、菱形(d)、五角星(p)、圆圈(o)、叉号(x)、星号(*)、点号(.)

等高线图

meshgrid可以建立独立变量的易用函数，可以产生矩阵元素


set告诉等高线之间的固定值
countour3命令把等高线化成三维的

sum函数，求一个数组的和

size命令检查每个数组的元素的个数

if-else 语句块，需要用end来结束

使用for循环编程

for循环是一个指令

计算标准差和中位数

std表示计算标准偏差

更多编程要点

format 设置一个变量来保存数据


这样matlab会输出：
请输入房子的总平方数：
disp命令，用来显示变量的值

while 语句

while是一个循环指令

switch 语句

代数方程求解与其它符号工具

解基本代数方程

solve命令解代数方程
第一种调用方法：


第三种调用方式：

二次方程求解

s(1)、s(2)可以调用两个根的值，当只有一个解时，就不必要了

符号方程绘图

ezplot可以产生我们输入的符号方程的图象

解方程组

solve解出x和y，我们要在调用它的时候快递两个参数给他


expand可以对方程进行展开

factor可以对一个或多个方程式进行因式分解


simplify可以进行多项式相除

taylor(f,m)表示求f的m项展开式

基本符号演算和微分方程

极限计算

limit命令计算极限

计算趋向无穷的极限

获得渐近线

hold on命令告诉Matlab要再添加一些材料

导数计算

diff可以计算符号导数


subs命令代入符号函数中的某个值

dsolve命令

dsolve解符号微分方程
dsolve('equ')

常微分方程(ODE)求解

dsolve命令可以求解ODE

方程组和相平面图

findobj命令告诉MATLAB查找图象中的线条，然后使用set命令为线条设置颜色
