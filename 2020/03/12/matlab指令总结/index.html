matlab指令总结

前言

  最初，我了解到matlab是由于参加数学建模竞赛的时候，在这学期的自动控制原理、信号与系统课程的时候会用到，所以原本我准备在上学的时候穿插着学一下matlab。但是在“传感器及检测技术”课程中，老师布置的作业里有一条需要使用matlab线性拟合，所以我就在寒假顺便把matlab看了一遍。
  我看了一本书–《matlab揭秘》郑碧波译，由于matlab本身是一个工具，所以指令有点多，不易记忆，所以，我准备写一个blog来记录一样，顺便总结一下，方便以后用的时候查询。
  前提说明一下，下面顺序和思路，都是仿照上面那本书所写，如果想学matlab可以自己看书学习，这里只是总结。

指令

MATLAB环境

命令窗口与算法基础

1 命令在”>>”提示符后面输入
2 如果未定义变量名，默认为ans

3 matlab输入基本算法：
两数相除，右除a/b，左除a\b=b/a
幂，a^b
相加，a+b
相减，a-b

赋值运算符

向量与矩阵

向量

向量(vector)一维数值数组。
1 列向量：
a=[2;1;4]
2 行向量：
v=[2 0 4] 或 v=[2,0,4]
3 转置 (‘)

从已存变量创建大向量

D=[A;B]

创建等差元素向量

x=[x:q:y] 或x=x:q:y
x为首元素，y为末元素，q为间隔
linspace(a,b,n)创建了a、b之间还有n个等差元素的向量
logspace(a,b,n)创建了10^a和10^b之间n个对数值等差的向量

特征化向量

命令length显示矩阵或向量最长的部分元素数目
max或min命令找出向量中数值最大和最小的元素
sum操作符可得到总和
sqrt可得到平方根
conj命令计算向量的共轭复数向量

向量的点乘和叉乘

dot(a,b)表示a、b两向量的点乘
cross(a,b)表示a、b两向量叉乘

引用向量元素

v(i)表示向量v的第i个元素
v(:)表示列出向量的所有元素
A(4:6)选出第四道第六个元素组成一个新的、含有3个元素的向量

矩阵相乘

A .*B 数组相乘
A * B 矩阵相乘

特殊类型矩阵

eye(n) 主对角线元素全为1其他元素全为0的方矩阵——单位矩阵

引用矩阵元素

A(m,n) 选出第m行n列的元素
A(:,i) 选出第i列所有元素
A(:,i:j)选出从第i列到第j列之间的所有元素

行列式与线性方程组求解

det(A) 计算矩阵A的行列式

例如：求解下面方程组
5x + 2y - 9z = -18
-9x - 2y + 2z = -7
6x + 7y + 3z = 29
解：

求矩阵的秩

rank(A) 表示矩阵A的秩

求逆矩阵与伪矩阵

inv(A) 可计算矩阵A的逆矩阵
pinv(A) 可计算矩阵A的伪逆矩阵

简化阶梯矩阵

rref(A) 使用高速-乔丹消元法产生矩阵A降阶后的阶梯形式
magic(n) 计算魔方矩阵是一个n*n矩阵，矩阵的元素从1到n平方之间，并且行元素的和等于列元素的和

矩阵分解

lu(A) 对矩阵A进行LU分解

绘图与图形

2D绘图基础

①在MATLAB中绘图包含下面三个步骤:
1.定义函数
2.指定要绘制的函数图形的值范围
3.调用 MATLAB的plot(x,y)函数
②[ start : interval : end ]表示stare<=x<=end,以interval递增
③通过xlabel,ylabel函数可以绘制你一个坐标轴的标签；通过title函数添加标题
④fplot 函数会绕过绘图的时间间隔，自动决定绘图的点数
fplot(‘function string’,[xstart,xend])
参数function string 表示要绘制图像函数，xstart，xend表示函数的区间

坐标轴命令

axis square 表示产生正方形的图像
axis equal 表示产生一个两坐标轴比例和间距相同的图像
axis auto 表示让matlab自己选择

在同一图像中显示多个函数

要绘制多个函数,只需调用plot(x,y)函数,其中参数使用一对对的”x,y”,”x y”与”x,y”之间相互独立,后面跟着用单引号引起来、用来表示我们所要绘制的第二条曲线风格的字符串.
如：plot(t,f,’:’t,g,’–’)
MATLAB图像中可以使用四种基本线条风格：
实线 ‘ - ‘ 虚线’ – ‘ 虚点线 ‘ -. ‘ 点线 ‘ : ‘

添加图例

legend加在plot命令后面，用单引号把你要添加为图例的文本引起来
legend(‘sinxh(x)’,’cosh(x)’)

设置颜色

MATLAB给用户绘制图像八种颜色选择
颜色 | 说明符
:-: | :-:
白色 | w
黑色 | k
蓝色 | b
红色 | r
青色 | c
绿色 | g
洋红 | m
黄色 | y

设置坐标比例

设置绘图范围，调用axis命令
axis([xmin xmax ymin ymax])

子图

subplot(m,n,p)表示产生的子图有m行和n列，p表示要贴上去的某个已经绘制的图形窗口

图像重叠和linspace命令

linspace命令用来产生x数集的新命令
linspace(a,b,n)表示在a和b间取出均匀分布的个点

极坐标和对数图像

polar是产生极坐标图像
polar( theta,r)
loglog 产生整个数据范围内的函数
semilogx产生图像x轴使用对数值，y轴仍然用直接值
semilogy产生图像y轴使用对数值，x使用直接值

离散数据绘图

bar 能用二维条形图把数据显示出来
stem 绘制成针头图
可以通过传递fill参数选项让MATLAB填充标记，其中包括方块(s)、菱形(d)、五角星(p)、圆圈(o)、叉号(x)、星号(*)、点号(.)

等高线图

meshgrid可以建立独立变量的易用函数，可以产生矩阵元素
meshgrid(-5L0.1:5,-3:0.1:3)
countour绘制等高线图
countour(x,y,z)
set告诉等高线之间的固定值
countour3命令把等高线化成三维的
countour3(z,n)
surface可以装扮图像

三维图像

mesh(x,y,z)产生三维图像
surf或surfc可以使图像带有渐变颜色
surfl可以显示三维光照物体的表面

shading interp/flat/faceted
flat是用同一颜色为每个网格进行着色并隐臧网格线
facted是显示网格线
interp则是使用颜色插值的办法着色

统计和MATLAB编程介绍

创建柱状图

bar(x,y)可产生柱状图，x、y为两个数组
bar3或bar3h来显示三维图象

基本统计

mean函数，求一个数组的均值
mean(a)
sum函数，求一个数组的和
sum(y)

编写MATLAB函数

创建可以在命令窗口中可以调用的函数:
我们要依次输入单词function、用来返回数据的变量名、函数名和用来传递数据的参数

size命令检查每个数组的元素的个数

if-else 语句块，需要用end来结束

使用for循环编程

for循环是一个指令

计算标准差和中位数

std表示计算标准偏差
std(raw)
median表示计算中位数
median(raw)

更多编程要点

format 设置一个变量来保存数据
format bank
input 命令通过指示符实时得到用户的数据
sqft = input (‘请输入房子的总平方数：’)
这样matlab会输出：
请输入房子的总平方数：
disp命令，用来显示变量的值

while 语句

while是一个循环指令

switch 语句
